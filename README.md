#### Running on local Machine :
1. set .env file in root folder
    1. add mongodb uri | MONGO_URI
    2. add jwt secret  | JWT_SECRET
2. root folder
    1. npm install
    2. npm run dev
3. cd frontend
    1. npm install
    2. npm run dev
